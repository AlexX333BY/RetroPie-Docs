## ROMS

Accepted File Extensions: **.d88 .88d .cmt .t88** 

Place your PC-88 ROMs in:
```
/home/pi/RetroPie/roms/pc88
```

## BIOS

Place your BIOS files in:
```
/home/pi/RetroPie/BIOS/pc88
```
